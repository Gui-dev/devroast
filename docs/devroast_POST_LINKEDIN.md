🛠️ Acabei de entregar um projeto open source completo: Devroast
Uma aplicação web que analiza código e dá feedback brutalmente honesto (ou construtivo, se preferir).
O que faz:
- Cola seu código e recebe uma "roast" sarcástica ou feedback honesto
- Scoring 0-10 baseado em qualidade, boas práticas e problemas potenciais
- Detecção de issues com níveis de severidade
- Sugestões de perbaikan geradas por IA
- Leaderboard para vergonha pública
Stack técnica:
⚡ Next.js 16 (App Router)
🔧 Fastify API (Hexagonal Architecture)
🐘 PostgreSQL + Drizzle ORM
🤖 Ollama (qwen2.5-coder) para análise de IA
📦 Turborepo monorepo
🎨 Tailwind CSS v4 + Design System
📚 Storybook com componentes documentados
✅ Testes unitários + E2E (Playwright + Supertest)
Diferenciais para recrutadores:
- Arquitetura hexagonal completa (routes → use-cases → repositories)
- TypeScript do início ao fim
- Storybook para documentação de componentes
- Testes unitários e E2E passando
- Clean Code e convenções rigorosas
